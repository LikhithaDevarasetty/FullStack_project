# Travel Booking Website  

A comprehensive travel booking website with user authentication, interactive maps, and booking management built with Python Flask and web technologies.  

<hr>

## Purpose  

To create an online platform that enables users to discover, plan, and book their travel experiences seamlessly. The website serves as a one-stop solution for travelers to explore destinations, make reservations, and manage their bookings efficiently.

## Features  

- <b>User Authentication System</b>: Secure login and registration with form validation  
- <b>Interactive Maps</b>: Visual representation of destinations and travel routes  
- <b>Destination Browsing</b>: Explore travel destinations with detailed information  
- <b>Booking Management</b>: Streamlined reservation process and booking dashboard  
- <b>Responsive Design</b>: Mobile-friendly interface using Bootstrap  
- <b>Form Validation</b>: Client-side validation using JavaScript and jQuery  
- <b>Dynamic Templates</b>: Server-side rendering with Jinja2

## Technologies Used  

### Frontend  
- **HTML5** – Semantic structure  
- **Bootstrap 5** – Responsive design and layout  
- **CSS3** – Styling and visual customization  
- **JavaScript** – Form validation and interactivity  
- **jQuery** – DOM manipulation and UI effects  

### Backend  
- **Python 3.x** – Core programming language  
- **Flask** – Lightweight web framework  
- **Jinja2** – Dynamic HTML rendering  
- **SQLite/MySQL** – Database for storing users and bookings
